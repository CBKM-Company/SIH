Why BlockChain ?

BlockChain gives power of ​ Immutability​ , Transparency​ , Accountability​ ,Security​ of transaction and assets. We are using private blockchain network As we cannot risk keeping the record in public blockchain because anyone can see the transaction . Although the data is Immutable but there will be privacy issue As BlockChain is ​ Immutable , ​ so no one can temper it and also the data is cryptographically hashed , linked to each other and stored so it’s difficult to decrypt it .

How do things work ?

We are using Hyper-Ledger fabric build by Linux foundation It helps in making public blockchain and various peers for RPC (Remote procedure Calls) As you can see in the first fig (Hyperledger Infrastructure ) We have 4 docker containers

  1. CA Server this provide public and private keys for every user and manage its expiration and all other things .MSP (Membership Service Provider )
  2. Order : as the name suggest that it keeps the track of sequence of transaction and we build it on KAFKA (it helps in making real time streaming pipelines which is s ​ calable​ )
  3. Peer : we are using at least 4 peers for development but it is more scalable every peer is connected to its own DataBase
  4. Couch DB :- this is what the hyperledger uses to store unstructured data . And deep down It uses LevelDB
  5. Client Peer is to provide RPC to client request
  6. Endorser peer​ checks the Certificate details and others to validate the transaction.
  7. Anchor peer​ receives updates and broadcasts the updates to the other ​ peers​ in the organization. As Anchor peer is discoverable so we can communicate to other organization by GOSSIP protocol
  8. There are two more folder C onfig ​ folder stores all the channel and peer configuration and ​ Crypto f ​ older stores all the crypto material like digital certificates

How are we maintaining security ?

We are storing millions of unstructured data mostly like text image pdfs ...etc Hence , We are ​ Using ​ asymmetric public-private keys​ provided by PKI (public key infrastructure ) from MSP (Membership service provider ) to encrypt-decrypt stored data. With RSA algorithm Just like SSL we are storing the data and enhancing the security And using blockchain we double up the security as data is stored in encrypted form and linked to each other through hash. Brings immutability
